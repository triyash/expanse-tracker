# Personal Expense Tracker

A beginner-friendly Personal Expense Tracker built using React (CDN), Chart.js and LocalStorage. No build tools required.

## How to run locally
1. Open the project folder.
2. Use Live Server extension in VSCode OR run: `python -m http.server 8000`
3. Open `http://localhost:8000` in your browser.

## Features
- Add / Edit / Delete income and expenses
- Categorization and filters
- Summary cards (Income / Expense / Balance)
- Category pie chart (Chart.js)
- Persistent storage using LocalStorage
- Export data as JSON

## Deploy (Vercel)
1. Push to GitHub (see commands in this repo's instructions).
2. Import the repository in Vercel and deploy (static site).
